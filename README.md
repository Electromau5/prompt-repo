# Prompt Repository

A prompt management application built with React, Vite, and Tailwind CSS.

## Features

- Hierarchical folder organization for prompts
- Search and filter prompts by tags
- Tag categories for organization
- Bulk import from JSON, CSV, Markdown, or plain text
- Backup and restore functionality
- Copy prompts to clipboard with one click
- Move prompts between folders
- Context menu for folder actions

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Lucide React (icons)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
